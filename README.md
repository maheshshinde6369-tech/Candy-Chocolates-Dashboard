# Candy Chocolates Dashboard Report

## Overview

The Candy Chocolates Dashboard was developed using Microsoft Power BI to analyze sales performance across products, factories, and divisions. The dashboard provides interactive insights using KPI cards, charts, maps, and slicers for better business analysis.

## Key KPIs

* Total Sales: 141.78K
* Total Quantity: 39K
* Total Cost: 48.34K
* Profit Margin: 65.91%

## Key Insights

* The Chocolate division contributes the highest sales.
* Products like Everlasting Gobstopper and Fizzy Lifting Drinks are top-performing items.
* Factory-wise analysis helps identify high-performing factories.
* Geographic visuals display factory locations and sales distribution.

## Tools & Technologies

* Power BI
* DAX
* Power Query
* Data Modeling

## Conclusion

This dashboard helps stakeholders monitor performance, identify profitable products, and support data-driven business decisions through a clean and interactive reporting interface.
